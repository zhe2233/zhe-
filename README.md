# Windows 风格弹窗爱心动画特效 ✨

基于 Canvas 实现的 Windows 风格弹窗爱心动画特效，由 **华为 CodeArts Agent AI** 辅助开发完成。

## 项目介绍
- 双击屏幕启动爱心成型动画，40个仿Windows弹窗拼成爱心形状
- 内置80+条暖心祝福文案，随机展示，适配表白、节日祝福场景
- 支持满屏散落交互、一键重置动画，响应式适配多设备
- 可自定义弹窗样式、文案内容、动画参数，轻松定制专属版本

## 功能特性
✅ 双击启动爱心成型动画
✅ 点击屏幕生成满屏散落弹窗
✅ 右下角一键重置动画
✅ 随机温馨祝福文案
✅ 仿Windows窗口拟态设计
✅ 响应式适配全尺寸屏幕

## 使用方法
1.  直接克隆仓库到本地：`git clone https://github.com/zhe2233/zhe-.git`
2.  用浏览器打开 `index.html` 文件即可查看效果
3.  可直接修改代码中的 CONFIG 配置项，自定义文案、样式、动画时长

## 技术栈
- 原生 HTML5 Canvas
- 原生 JavaScript（无第三方依赖）
- CSS3 渐变与动画
